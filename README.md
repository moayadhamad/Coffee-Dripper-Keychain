# Coffee-Dripper-Keychain
This project is a coffee dripper keychain designed in Onshape by tracing an image from the internet. The model includes a 4 mm hole for a keychain and was extruded to a thickness of 2 mm.
## Software
- Onshape

## Features
- Coffee dripper design
- 4 mm keychain hole
- 2 mm extrusion
- Exported as STL
